Programme relais entre la souris et l'ordinateur, 
utilisant une carte Leonardo, USB shield et IR Receiver. 

Le code d'activation du relais souris -> ordi est reçu par le IR Receiver, branché sur la PIN 2.

Le moniteur série doit être activé, sur la channel 115200.

Le code que l'on a écrit est dans le dossier main, les autres dossiers sont
ceux des librairies utilisées: 
https://github.com/felis/USB_Host_Shield_2.0
http://github.com/shirriff/Arduino-IRremote
